# Juriprazo_backend

### Projeto de Desenvolvimento para o Trabalho de Conclusão de Curso.

#### Projeto focado no desenvolvimento de um sistema web para o controle e monitoramento de prazos jurídicos. O objetivo principal é automatizar a gestão de datas críticas para escritórios de advocacia e departamentos jurídicos, reduzindo riscos de perda de prazos e aumentando a eficiência operacional.
